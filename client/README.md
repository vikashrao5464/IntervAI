# IntervAI Client

This folder contains the React frontend for the interview practice app.

## What is inside

- `src/components` for reusable UI and interview flow steps.
- `src/pages` for route-level screens.
- `src/redux` for the global user state.
- `src/utils` for helper integrations.
- `public` for static assets.

## Development

Install dependencies from this folder and run the dev server:

```bash
npm install
npm run dev
```

The main project documentation lives in [../../README.md](../../README.md).
